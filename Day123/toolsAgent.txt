⚡ Tools ka kaam

👉 Tools AI ko real-world actions karne dete hain:

Tool	AI kya karega
🔍 Search Tool	Internet pe search karega
🧮 Calculator Tool	Maths solve karega
🗄️ Database Tool	Data fetch karega
🌐 API Tool	External service call karega
🧠 Example samajh lo
Without tools ❌

User: “India ki current population kya hai?”
AI: Guess karega (old data)

With tools ✅

AI:

Decide karega → "mujhe search karna hai"
Tool use karega → Google/API
Fresh data laayega
Accurate answer dega
