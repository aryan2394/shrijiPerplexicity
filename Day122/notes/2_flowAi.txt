📌 Image mein kya ho raha hai:
User (tum) → “Hi” bhejta hai
Backend / Server → request ko receive karta hai
AI Service Provider (ASP) → jaise OpenAI, Gemini etc. ko request bheji jaati hai
AI Response → “Hello, how can I help you today?” wapas aata hai
Server → User → final response tumhe dikhta hai

🧠 Simple Flow:
User → Backend → AI API → Backend → User
🔥 Important Concept:
{ role: "user", content: "Hi" } → user ka message
{ role: "AI", content: "Hello..." } → AI ka response
